# Privacy Policy

**Last Updated:** November 20, 2025
**Version:** 3.3.0

## Overview

This privacy policy describes how "AI Turkish Text Corrector" Chrome Extension ("Extension", "we", "our") handles user data. By using this Extension, you agree to this privacy policy.

## Data Collection and Storage

### Locally Stored Data (On Your Device Only)

The following data is stored **only on your device** using Chrome's local storage mechanism and is **never transmitted to us**:

- **OpenAI API Key**: Stored in `chrome.storage.local`
- **Custom System Prompt**: Stored in `chrome.storage.local`
- **Extension Settings**: Enable/disable state stored in `chrome.storage.sync`

**We do not have access to, control over, or store any of this data on our servers.**

### Data Sent to Third Parties

When you click the "Correct" button:

- **Your text content** is sent directly from your browser to OpenAI's API
- This transmission occurs between your device and OpenAI's servers
- We do not intercept, store, or have access to this data

## What We Do NOT Collect

- ❌ Personal identification information
- ❌ Browsing history

- ❌ Analytics or telemetry
- ❌ Location data
- ❌ Any data sent to our own servers (we have no servers)

## Data Usage

The Extension's sole purpose is to correct Turkish text according to TDK standards. We do not:

- Sell or share user data with third parties
- Use data for advertising purposes
- Perform any analytics or tracking
- Store user data on any server we control

## User Rights

You have the right to:

- **Delete your data**: Uninstall the Extension to remove all local data
- **Manage your data**: Access Chrome extension settings to view/modify stored data
- **Opt-out**: Simply do not use the Extension

## Security

- All API requests use HTTPS encryption
- We implement no additional security measures as we do not collect or store any user data
