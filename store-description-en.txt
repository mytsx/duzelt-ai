## Supported Editors

- CKEditor 4.x and 5.x
- Summernote
- TinyMCE
- Quill

## Important Note

This extension works ONLY with the rich text editors listed above. It does NOT work with plain text areas like Gmail or LinkedIn. It is designed for professional content management systems.

## Privacy

- Your API key is stored only on your device
- Your texts are sent to OpenAI API (OpenAI privacy policy applies)
- The extension does NOT collect or send any data to third parties
- Open source - you can review the code on GitHub
